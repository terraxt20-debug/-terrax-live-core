# ⚽ TerraX Live - Core Streaming & Matches Repository

هذا المستودع هو القلب النابض والمسؤول الحصري عن إدارة وتغذية قنوات البث المباشر، جداول المباريات اليومية، والتصنيفات الهيكلية لتطبيق **TerraX Live**. يتم استدعاء ملفات هذا المستودع ديناميكياً بواسطة Supabase Edge Function (`smooth-service`) عبر مسار الجلب الفريد لضمان استقرار البث وسرعته الخيالية.

---

## 📂 الهيكل الشجري للمستودع (Repository Structure)

يجب الحفاظ على تنظيم الملفات داخل هذا المستودع بالشكل التالي لضمان دقة الجلب البرمجي وعدم حدوث أخطاء مسارات:

```text
terrax-live-core/
├── beinsport.m3u                  # الملف المركزي لروابط قنوات البث المباشر المشفرة والحية
├── gamePlaySchedule.json          # الجدول الذكي لتحديث تفاصيل ومواعيد المباريات اليومية
├── categories.json                # تصنيفات واجهة التطبيق الرئيسية (رياضة، قنوات حية)
├── groupStage.json                # بيانات ترتيب المجموعات والبطولات الحالية
├── FIFA-World-Cup-Attendance.json # ملف بيانات حضور كأس العالم المدمج
└── README.md                      # ملف التوثيق والتعليمات الحالي
